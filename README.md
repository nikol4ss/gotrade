## GoTrade

An application that consumes financial market APIs, provides analytical dashboards, and simulates investments using AI/ML.

### About

This project is a financial application that integrates data from market APIs to provide real-time analytics and insights. It features interactive dashboards for visualizing market trends and performance metrics. Additionally, it uses AI and machine learning models to simulate investment strategies, helping users explore potential outcomes and optimize decisions for real investments. The system combines data integration, predictive analytics, and visualization to support informed financial planning and strategy testing.
